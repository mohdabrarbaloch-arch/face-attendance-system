\# Face Attendance System



This project is a Face Recognition based Attendance System built using Python.

It detects faces through a webcam and automatically marks attendance.



\## Features

\- Real-time face detection

\- Face recognition using OpenCV

\- Automatic attendance marking

\- Stores attendance in CSV file



\## Technologies Used

\- Python

\- OpenCV

\- NumPy



\## How to Run

1\. Install required libraries

2\. Run the main Python file

3\. Show face to camera

4\. Attendance will be marked automatically
